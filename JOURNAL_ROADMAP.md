# The Cure: Journaling Experience Roadmap 2026

## Vision
Transform journaling into a sacred, AI-guided dialogue between the user and their own wisdom—where technology becomes a compassionate mirror for self-discovery.

---

## Phase 1: MVP (Weeks 1-4) — Intelligent Journaling Core
### Core Features
- **Emotional Prompting System**
  - AI generates contextual prompts based on time of day, mood, intention
  - 3 prompt levels: light, medium, deep introspection
  - Users can customize prompt themes (gratitude, healing, creativity, vision)

- **Smart Entry Writing**
  - Rich text editor with emotional tone detection
  - Real-time AI elaboration suggestions ("expand on this thought")
  - Voice-to-text integration (future: emotion-aware transcription)
  
- **Mood & Energy Tracking**
  - Mood slider (1-10) with symbolic representation (• ◯ △)
  - Energy level indicator
  - Body sensation check-in (somatic awareness)
  
- **Intelligent Archiving**
  - Date-based timeline view
  - AI-generated tags (emotion, theme, insight)
  - Searchable by mood, keyword, or emotional state

- **Reflection Insights**
  - Weekly summaries of emotional patterns
  - Recurring themes detected by AI
  - Growth tracking with Light Points

### Tech Stack
- React components with atomic architecture
- Mock AI service (ready for OpenAI integration)
- LocalStorage for MVP persistence
- Vitest for TDD

## Success Metrics
- Users complete ≥5 entries/week
- Average session: 10-15 minutes
- AI prompts rated 4.5/5 for relevance
- 80% return rate weekly
- Light Points earned through consistent practice

---

## Design Principles
1. **Sacred, not surveillance** — Journal is user-owned; no tracking beyond consent
2. **Elaboration vs. analysis** — AI helps you think deeper, not judge
3. **Symbolic coherence** — Every interaction aligned with • ◯ △ system
4. **Emotional safety** — Tone detection prevents harmful suggestions
5. **Embodied journaling** — Include somatic prompts, voice, sensory input

---

## User Journey
1. Open journal → see morning prompt from Elayra
2. Check-in: mood, energy, body sensation
3. Free-write or guided flow
4. AI suggests elaborations on key thoughts
5. Tag emotions and insights
6. End with gratitude or intention
7. Earn Light Points, see reflection insights
8. Archive entry with emotional metadata
